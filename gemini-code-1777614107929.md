# JobHunter: Persistent Candidate Search Assistant

## Overview
JobHunter is a stateful, long-running career co-pilot built entirely on LangGraph and Streamlit. It manages a job seeker's full application lifecycle across multiple sessions. It actively scans the user's Gmail for rejection feedback, uses an LLM to detect hidden skill gaps, and autonomously proposes strategic pivots to improve the candidate's success rate.

## Problem
Active job seekers apply to 50-100 roles a month, managing the chaos in static spreadsheets. They rarely synthesize rejection feedback to change their strategy, leading to burnout. Additionally, creating profiles on job boards is tedious and manual.

## Solution
An intelligent state-machine agent that acts as a persistent memory bank. JobHunter:
1. **Magic Onboarding:** Uses an LLM to read a candidate's uploaded CV and instantly auto-generates their search strategy, target salary, and base profile.
2. **Dynamic Matching:** Matches candidates to local job datasets based on a dynamic "Search Strategy".
3. **Automated Tracking:** Scans the candidate's Gmail inbox to track application outcomes and parse rejection feedback.
4. **Strategy Pivot:** When a rejection pattern is detected, it pauses the workflow to suggest a data-driven pivot to the search strategy.

## Target Users
* High-volume tech job seekers applying to 10+ roles per week.
* Career pivoters struggling to find the right seniority fit in a tough market.
* Candidates who are consistently failing at specific interview stages.

## MVP Scope
* "Magic Onboarding" via PDF Resume upload and LLM extraction.
* Multi-session persistence using LangGraph's `SqliteSaver` checkpointer.
* Job feed powered by a local CSV/Kaggle dataset to ensure demo stability.
* Real-time Gmail API integration (OAuth) to fetch employer emails.
* LLM-driven pattern detection engine (requires 3 historical data points to trigger a shift).
* Three-panel Streamlit dashboard: Job Feed, Application Tracker, and Agent Strategy Log.
